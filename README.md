# 永續半導體發展路線圖網站

## 📌 專案簡介

這是一個專注於半導體產業永續發展的靜態網站。展示ESG（環境、社會、治理）理念與減碳目標的視覺化路線圖。

## 🎯 核心目標

- **2025年**：建立碳排放基準線
- **2030年**：實現碳排放減少 50%
- **2035年**：實現碳排放減少 70%
- **2050年**：實現淨零排放

## 📁 專案結構

```
├── index.html      # 主頁面
├── style.css       # 樣式表
├── script.js       # 互動功能
└── README.md       # 說明文件
```

## 🎨 設計特點

- **現代化設計**：採用最新的 CSS Grid 和 Flexbox 布局
- **響應式設計**：完美適配桌面、平板和手機屏幕
- **色彩方案**：採用綠色系（代表環保）主題
- **互動效果**：懸停動畫和平滑過渡
- **可訪問性**：語義化 HTML 標記

## 🛠️ 技術棧

- **HTML5** - 結構化標記
